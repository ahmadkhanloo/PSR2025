🧩 Why Join?

If your files look like this: task_final.py task_final2.py task_final_really_final.py – then this workshop is for you!

A 3-day hands-on workshop on Git + Python for cognitive researchers.

  • ✅ Use Git & GitHub for collaboration
  • ✅ Write clean, modular Python code
  • ✅ Manage projects efficiently
  • ✅ Testing & documentation habits
  • ✅ Turn research code into pro assets

کارگاه **۳ روزه عملی** Git + Python برای **محققان شناختی**.

  • ✅ Git و GitHub برای همکاری
  • ✅ کد Python تمیز و ماژولار
  • ✅ مدیریت پروژه کارآمد
  • ✅ تست و مستندسازی
  • ✅ تبدیل کد تحقیق به دارایی حرفه‌ای

  • Want cleaner, shareable, reproducible projects
  • Minimal experience → level up workflow!

🧑‍🔬 برای کیه؟

  • می‌خواید پروژه‌هاتون تمیز، قابل اشتراک و تکرارپذیر بشه
  • تجربه کم؟ → workflow رو **لول آپ** کنید!
